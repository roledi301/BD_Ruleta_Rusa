Las ruletas es uno de los principales juegos del Casino.
En cada tirada que se produce en una ruleta se presentan de forma gráfica en una pantalla de usuario todos los datos relativos a la 
fase de juego, al número ganador, y a las estadísticas de las últimas tiradas. Además en la pantalla de cliente aparecen datos informativos
adicionales como el nombre del crupier en cada momento, y mensajes publicitarios.
Una tirada es válida cuando el plato y la bola giran en sentido contrario (uno clockwise, y otro anticlockwise); tanto el plato como la
bola dan un mínimo de 5 giros antes de caer la bola en ningún número; entre 2 tiradas consecutivas, se cambia siempre el sentido de giro
tanto de la bola como del plato. Si no se cumple alguna de estas condiciones, el sistema avisa de “partida nula”, no mostrando ningún
número ganador.
La ruleta aporta cierta información y es muy útil a la hora de detectar un posible fallo en cualquiera de ellas o incluso fraude. 
Esta información se consigue gracias a que los cilindros que componen las ruletas están equipados con sensores ópticos capaces de
detectar en tiempo real los parámetros de funcionamiento: Velocidad, conteo y sentido de giro tanto del plato y como la bola, e 
identificación del número ganador en
cada tirada. Cada ruleta se coloca sobre una mesa y se conecta mediante un cable de datos con una “caja de control” que transmite 
la información a un ordenador de control para el registro de los datos.
El Casino está dotado de multitud de bases de datos, como cualquier otra empresa, para poder gestionar y llevar un control de todo 
el entorno. Exceptuando el caso de las ruletas, que aún almacenan sus datos en ficheros planos que conlleva bastante tiempo, por lo
que apenas se usa esa información.

El número ganador 45 no existe en la ruleta (el plato tiene 37 números, entre el 0 y el 36, ambos inclusive). El registro con número
ganador 45 no debe ser tenido en cuenta: Indica un cambio de crupier. El siguiente registro ya viene informado con un nuevo id y 
nombre de crupier.
El desarrollo de una aplicación basada en el uso estadístico de estos datos puede proporcionar al Casino una importante herramienta
para mejorar el nivel de control y seguridad sobre la actividad en las ruletas, tanto de jugadores como de empleados.
Se plantea el desarrollo de una base de datos para conseguir que haya un acceso mucho más directo a la información de las ruletas, 
y aprovechar esta información al máximo.
